# [📋 Trello Clone (Client)](https://trello-m0b.vercel.app/)

A high-performance Kanban board implementation focused on seamless drag-and-drop interactions and a modern developer experience.

## 🛠️ Tech Stack

- **Core:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** TailwindCSS
- **drag-and-drop:** dnd-kit

## 🛠️ Getting Started

### 1. Clone & Enter

```bash
git clone https://github.com/mind0bender/trello.git
cd trello/client
```

### 2. Install Dependencies

> I recommend using Bun for the fastest installation, but standard package managers work too.

```bash
bun i         # Recommended
# or npm i / yarn
```

#### Spin up Development Server

```bash
bun dev        # Recommended
# or npm run dev / yarn dev
```

🚀 The application will be live at > [http://localhost:5173/](http://localhost:5173/)
